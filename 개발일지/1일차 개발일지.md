# 디지몬 어드벤처 월드 개발일지

## 💡 금일 작업 내용
- [1] **[처음 로딩창 기능]** 게임 처음 로딩창이자 홈 화면 설계 
- [2] **[메뉴바 기능]** 편의 기능 추가 
- [3] **[수정]** 버그 수정 및 리팩토링 내용

### [1] **[처음 로딩창 기능]** 
- **기능:** 유저가 디지털월드로 들어가는 느낌을 주기 위해서 화면에 효과를 넣었고 디지털 월드 게이트 진입이라는 버튼을 넣어 버튼을 눌렀을때 들어갈 수 있게 만들었다
- **발생 문제:** 효과가 화면전체에 있어야되는데 계속 화면 중심부에만 효과가 있었으나 점차 수정중이다.
- **보안점:** AI가 이해할 수 있게 정확하게 설명이 필요할것같다. 추가로 BGM이 필요할 것 같다.

### [2] **[메뉴바 기능]** 
- **기능:** 작은 메뉴바 형태를 만들었고 처음화면으로 돌아가는 기능을 넣었다.
- **발생 문제:** 아직은 문제가 없으나 로그인 기능을 추가하면 내부에서 충돌이 있을 것 같아서 이후에 보안점이나 개선점을 생각해야겠다.
- **보안점:** 로그인 기능과 같은 편의성 옵션을 넣을 예정이다.
